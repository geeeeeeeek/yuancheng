# 全栈开发工程师

# 岗位描述

我们是一家致力于区块链技术创新的公司，正在招募具有丰富经验的区块链前端或全栈开发人员。如果你希望在区块链行业大展身手，并获得每小时 20-40 USDT 的薪酬，欢迎申请加入我们的团队。

职位要求：

前端开发：

熟练使用 Next.js ，能够构建高性能、响应迅速的 Web 应用。 熟悉 shadcn 或其他现代前端 UI 库，具备良好的用户体验设计思维。

数据库管理：

熟练掌握 Subgraph 框架，能够从链上数据中提取并构建查询接口。 精通 PostgreSQL 数据库设计与优化，能够处理复杂查询和大规模数据集。

智能合约交互：

具备与 智能合约 进行交互的经验，理解合约部署和调用的底层机制。

加分项：

具备智能合约开发能力，熟悉 Solidity 或其他智能合约开发语言。 拥有网站运营经验，能够参与和协助产品的运营优化。

工作内容：

参与和开发我们区块链项目的前端/全栈部分，优化用户体验。 维护与数据库及链上数据的交互接口，确保系统稳定高效。 配合团队，参与智能合约相关功能的前端集成与互动。

# 薪资

20-40 USDT/小时
