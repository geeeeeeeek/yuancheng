# 中级全栈工程师

# 岗位描述

我最近在组建一个新团队负责 Bitcoin Mining 方向的业务，希望在此招聘 2 位工程师冷启动，团队分布在新加坡、北美、其他地区等，主要解决 Mining 流程自动化和数据挖掘等工作。

业务合规，你不必担心常见的该行业合规顾虑

法币薪酬，你不必烦恼常见的 Stablecoin 问题

远程友好，你也可以选择 base 新加坡

高级工程师/ 中级工程师 (25 - 45K) * 2

要求

全栈工程师，你可以擅长前端，也可以是擅长后端业务，这都可以，我们主业务基于 Typescript ，其他技术也不受限，最重要的是交付高质量的产品。

有良好的技术品味，使用简单朴素的方式解决问题，掌握常见的 Web 开发最佳实践，也熟悉数据仓库 ETL 设计与流程，有独立动手和负责业务能力最好了。

期待你对区块链 / Bitcoin 感兴趣，对其工作原理有所了解，如果能深入 Bitcoin 节点进行定制化修改和优化那就更好了，这样我们有更多共同话题，也免去学习门槛。

习惯远程工作与生活的节奏，能为自己提供一个安静、可持续的工作环境，免于家庭事务的打扰最佳，远程工作是 work life balance ，但也容易被生活琐事干扰。

其他

简历，最好提供 Github / Blog 等辅佐信息

在入职前，一般我会按你的时薪支付 1 - 2 周的工作，互相磨合与评估是否匹配。

(25 - 45K) * 2

新加坡

远程
