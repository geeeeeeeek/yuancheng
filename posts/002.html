# PHP开发工程师

# 岗位描述

负责系统服务端设计和开发，参与项目需求分析、系统设计、测试用例评审等

参与系统架构、性能、安全、扩展等设计和实现及相关优化等

根据产品及相关需求，积极配合相关人员，按时完成产品迭代上线

任职要求:

1.计算机及相关专业本科及以上学历，具有 5 年以上 PHP 开发经验 2.PHP 基础扎实，精通 Phalcon ，laravel, Yii 等框架，对生产系统故障排查有实际工作经验 3.有大流量并发场景的开发经验，熟悉多进程框架 Swoole 、Workman 有了解 4.精通 Mysql 、mongoDB ，Hbase 等数据库，对数据库有较强的设计能力,对 MySQL 有较深入的认识 具有索引优化、查询优化和存储优化经验、熟练掌握关系数据库理论 5.精通 LNMP 环境及 Nginx 及 Php-fpm 的配置和优化, 能独立搭建服务器环境以及项目线上部署 6.精通 Redis 、Memcache 缓存,对其使用场景及其限制有深入认识 7.精通 ElasticSearch 全文搜索引擎,Zookeeper,RabbitMQ,Kafka 等的使用 8.思维活跃，逻辑清晰，善于思辨和抽象，具备良好的沟通表达和团队协作能力 9.有分布式、集群、高并发、高负载、高可用系统或大型网站开发经验者优先 10.有一定 java 开发经验，能快速学习并掌握 Spring ,zookeeper ，kafka ，hbase 11.自我驱动力强，有强烈的事业心和责任感，结果导向，具备良好的协调推动能力和抗压能力"
