# React 前端开发工程师

# 岗位描述

参与 Next.js 14 框架升级, 从 Pages 目录迁移到 App 目录

参与网站前端的业务模块开发，根据 Core Web Vitals 持续优化用户端体验。

我们主要面向海外用户，所以需要熟悉 Google 搜索引擎的 SEO 规则。

谷歌每段时间都会更新 搜索排名规则，前端需要持续关注谷歌的规则变化日志，并及时更新网站中所需要调整的前端部分。

对待工作要求认真、负责，合理的规划任务的排期，遇到问题需要及时沟通，忌讳交付任务时才反馈任务问题。

基本要求

计算机或相关专业本科全日制及以上学历，至少有 4 年以上的工作经验。

熟练使用 React 、Next.js 、Typescript 、TailwindCSS 。

熟悉第三方组件库和工具 Zod 、SWR 、Zustand 、React Aria 、Shadcn UI 。(目前都在用，需要熟练，不熟悉的进来后上手比较慢)

熟悉 Core Web Vitals 指标和 Google SEO 的优化。

对前端工程化、模块化和组件封装有深入理解和实践

熟悉语言特性，依赖管理机制，了解语言发展近况。

有前端性能优化、加载体积、加载速度等，优化经验者优先。

20k ~ 28k

国内
