# 前端开发工程师

# 岗位描述

(20~35K 人民币，优秀者可谈)

1.负责公司产品的前端开发和维护

2.负责相关产品的需求以及前端程序的实现，提供合理的前端架构，实现前端组件化开发

3.使用 web 端进行 AVG 游戏开发，及 AVG 游戏编辑器开发。

4.与设计师和后端工程师紧密合作，确保最终产品的品质

技能要求

3 年以上前端大型/中型 C 端网站开发经验

熟悉 JavaScript 、CSS 、React ，vue ，electron js ，ADV js 等前端基础技能

掌握 web 前端性能优化方法，精通 JS 、网络通信、浏览器等底层原理

熟悉 NodeJS 服务开发，具备一定全栈开发能力

具备低代码平台或策略性游戏开发经验优先

有良好的团队合作精神和沟通能力
