# 全栈开发工程师

# 岗位描述

作为全栈开发工程师，您将远程工作，主要负责从前端到后端的 Web 应用程序开发。您将与我们的 AI 工程师、产品经理及其他团队成员紧密合作，确保项目的顺利实施与交付。

工作职责

设计、开发和维护基于 React 的用户界面。

使用 Node.js 和 NestJS 构建高效、可扩展的后端服务。

参与系统架构设计，确保代码质量和系统安全。

优化应用程序性能，处理跨浏览器兼容性问题。

参与代码审查和持续集成，推动开发流程的优化。

与团队成员合作，解决开发过程中遇到的问题。

职位要求

熟练掌握 React ，能够独立开发复杂的前端应用。

深入理解 Node.js ，熟悉常见的 Node.js 框架，尤其是 NestJS 。

有全栈开发经验，能够处理从前端到后端的完整开发流程。

熟悉 RESTful API 设计，了解 GraphQL 更佳。

掌握前端和后端的性能优化方法。

良好的团队合作精神和沟通能力，能够与多方协作推动项目进展。

良好的问题解决能力，能够在压力下高效工作。
