# 全栈(偏后端)

# 岗位描述

负责项目的框架设计与开发，确保代码质量和性能； 负开发生命周期中的所有阶段； 与团队成员远程协作，了解产品需求，并进行技术选型和方案设计； 持续关注新技术和发展趋势，提高产品质量和用户体验。

任职要求：

熟练掌握 React 或者 Vue 及其生态圈，具备扎实的 JavaScript 、HTML 和 CSS 基础知识；

熟练掌握 PHP, Python or Go 等等任意后端语言；

至少 3 年开发工作经验，具备良好的编程习惯和代码规范；

适应远程工作模式，具备良好的自律和自驱力。

加分项：

熟悉 Laravel ； 熟悉 Docker;

熟悉物流行业；

LLM / RAG / AI Agent / Machine Learning 等任一方向有实际落地经验者优先。

薪资福利:

薪资税前 18k - 25k （远程岗位国内五险一金费用需要自己缴纳）； 周末双休+法定节假日。 工作方式和语言： 全职远程，北京时间白天工作 8 小时，灵活合理安排时间，协作和开会时必须在线。

语言：中文。相关文档的阅读会要求英文但无需听说，研发团队均用中文沟通。
